Input: a grading \(\mathcal V\text{.}\) Output: a universal realization \(\widetilde {\mathcal V}\) of \(\mathcal V \text{.}\)

  1. Let \(\Omega = \{\alpha_1,\dots,\alpha_n \}\) be the set of weights of \(\mathcal V\) and let \(\{e_1,\dots,e_n\}\) be the canonical basis of \(\mathbb{Z}^n\text{.}\) Set \(R = \emptyset\text{.}\)
  2. Repeat for all pairs \(\alpha_i, \alpha_j \in \Omega\text{:}\) If \([V_{\alpha_i}, V_{\alpha_j}] \neq 0\text{,}\) extend \(R\) by \(e_i + e_j - e_k\text{,}\) where \(k\) is the index such that \(\alpha_k = \alpha_i+\alpha_j\text{.}\)
  3. For all \(i=1,\dots,n\text{,}\) set \(\widetilde V_{\pi(e_i)} = V_{\alpha_i}\text{,}\) where \(\pi \colon \mathbb{Z}^n \to \mathbb{Z}^n/\langle R\rangle\) is the projection. Return the obtained \(\mathbb{Z}^n/\langle R\rangle\)-grading.
