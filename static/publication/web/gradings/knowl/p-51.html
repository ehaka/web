Input: a grading \(\mathcal V\) that has a realization. Output: a universal realization \(\widetilde {\mathcal V}\) of \(\mathcal V \text{.}\)

  1. Let \(\Omega = \{\alpha_1,\dots,\alpha_n \}\) be the set of weights of \(\mathcal V\) and let \(B = \{e_1,\dots,e_n \}\) be the canonical basis of \(\mathbb{Z}^n\text{.}\) Set \(R = \emptyset\text{.}\)
  2. Repeat for all pairs \(\alpha_i, \alpha_j \in \Omega\text{:}\) If \(0 \neq [V_{\alpha_i}, V_{\alpha_j}] \subseteq V_{\alpha_k} \) for some \(\alpha_k \in \Omega\text{,}\) extend \(R\) by \(e_i + e_j - e_k \text{.}\)
  3. For all \(i=1,\dots,n\text{,}\) set \(\widetilde V_{\pi(e_i)} = V_{\alpha_i}\text{,}\) where \(\pi \colon \mathbb{Z}^n \to \mathbb{Z}^n/\langle R\rangle\) is the projection. Return the obtained \(\mathbb{Z}^n/\langle R\rangle\)-grading.
