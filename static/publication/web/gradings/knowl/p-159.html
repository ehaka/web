In this section, we present how knowing a maximal grading of a given nilpotent Lie algebra \(\mathfrak{g}\) can be used to determine a list of Heintze groups over \(\mathfrak{g}\text{.}\) However, note that when working in the non-algebraically closed field \(\mathbb{R}\text{,}\) we cannot in general obtain the maximal grading using Algorithm 3.11, see Remark 3.15.
