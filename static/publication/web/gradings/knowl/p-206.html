Input: A grading \(\mathcal{V}\) of a Lie algebra \(\mathfrak{g}\text{.}\) Output: Decision if \(\mathcal{V}\) admits a positive realization.

  1. Form the \(N \times k\) matrix \(A\) associated with the problem and set \(\mathbf{b}=(-1,\ldots,-1)\in\mathbb{R}^{N}\text{.}\)
  2. For each invertible \(N\times N\) matrix \(B\) formed from the columns of the matrix \(A\) do the following: Compute \(\mathbf{x}=B^{-1}(\mathbf{b})\text{.}\) If \(\mathbf{x} \ge 0\text{,}\) then the grading admits a positive realization.
  3. Otherwise, the grading has no positive realization.
