Algorithm 3.8. Positive realization.

Input: A torsion-free grading \(\mathcal{V}\) for a Lie algebra \(\mathfrak{g}\) over a computable field. Output: A positive realization of \(\mathcal{V}\) or the non-existence of one.

  1. Compute the universal realization \(\widetilde{\mathcal{V}}\) of \(\mathcal{V}\) using Algorithm 2.8. Let \(\alpha_1,\ldots,\alpha_N \in \mathbb{Z}^k\) be the weights of \(\widetilde{\mathcal{V}}\text{,}\) let \(A_0 \) be the \(N \times k \)-matrix whose rows are the vectors \(\alpha_i \in \mathbb{Z}^k \text{,}\) and let \(A \) be the block matrix of size \(N \times 2k \) defined as \([A_0 \:| {-}A_0] \text{.}\)
  2. Check if the linear programming problem \(A y \ge 1 \text{,}\) \(y \ge 0\) has a solution \(y \in \mathbb{Q}^{2k}\text{.}\) If not, return that \(\mathcal{V}\) does not have a postive realisation. Otherwise, let \(z,z' \in \mathbb{Q}^k\) be such that \(y=(z,z')\) and define \(w = z-z'\text{.}\)
  3. Check if the homomorphism \(f\colon \mathbb{Q}^k\to\mathbb{Q}\text{,}\) \(f(\cdot)=\langle w,\cdot\rangle\) is injective on the weights \(\alpha_i\text{.}\) If yes, return the push-forward grading \(f_*\widetilde{\mathcal{V}}\text{.}\)
  4. Let \(i\neq j\) be a pair of indices such that \(f(\alpha_i)=f(\alpha_j)\text{.}\) Set
    \begin{gather*} \eta=\varepsilon \delta \frac{\alpha_i-\alpha_j}{\norm{\alpha_i-\alpha_j}} \qquad \text{with} \qquad \varepsilon = (1+ \max\{\norm{\alpha_i}_{\ell^\infty}\})^{-1} \\ \text{and} \qquad \delta=\frac{\min\{ \braket{w,\alpha_h-\alpha_k} : \braket{w,\alpha_h-\alpha_k} \gt 0 \}}{4 \max \{\norm{\alpha_h-\alpha_k}_{\ell^\infty}\}}\text{.} \end{gather*}
  5. Set \(c = (\min_i\{\braket{w+\eta,\alpha_i}\})^{-1}\text{.}\) Now redefine \(w=c(w+\eta)\) and go back to step 3.
Proof of correctness.
