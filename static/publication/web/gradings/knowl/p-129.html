If the base field is not algebraically closed, finding diagonalizable elements of \(C(\mathfrak{t})\setminus\mathfrak{t}\) is more difficult. The subset of diagonalizable elements is a semialgebraic set in \(\mathfrak{gl}(\mathfrak{g})\text{,}\) and constructions to extract points from such sets exist, see for instance SectionÂ 13 of [1] on the existential theory of the reals. The problem is that these methods are practical only in low dimensions, whereas \(\dim\mathfrak{gl}(\mathfrak{g})=\dim(\mathfrak{g})^2\text{.}\) For Lie algebras defined over finite fields, more efficient randomized algorithms to find split tori exist, see [5] and [29].
