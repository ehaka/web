Item 4.
For all \(i=1,\dots,N\text{,}\) set \(\widetilde V_{\pi(e_i)} = V_{\alpha_i}\text{,}\) where \(\pi \colon \mathbb{Z}^N \to \mathbb{Z}^N/\langle R\rangle\) is the projection. Return the obtained \(\mathbb{Z}^N/\langle R\rangle\)-grading. Algorithmically, this step is equivalent to reducing a matrix to its Smith Normal Form. For an estimate on its computational efficientcy, we refer to [Storjohann].
