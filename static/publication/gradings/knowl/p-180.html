Applying the above equivariance to the definition (4.3) we find that the map \(\tilde{\phi}\) is \(D_\lambda\)-homogeneous, i.e., \(\tilde{\phi}(\lambda x) = D_\lambda(\tilde{\phi}(x))\) for all \(x\in\mathbb{R}^n\) and \(\lambda\gt 0\text{.}\) Consequently the map \(\tilde{\phi}\) is surjective if and only if it is open at zero. Since the change of parameters in (4.3) is a homeomorphism, the same is true also for the map \(\phi\text{.}\)
