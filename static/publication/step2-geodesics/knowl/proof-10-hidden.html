The curve \(a\colon[0,T]\to V_1^*\) will be given by restricting the linear map

\begin{equation} a(t) := (L_{\gamma(t)})^*\lambda(t)\colon \mathfrak{g}\to\RR\tag{3.13} \end{equation}

to \(V_1\text{.}\) The skew-symmetric bilinear form \(B\colon V_1\times V_1\to\RR\) will be given by

\begin{equation} B(X,Y) = a(t)[X,Y]\text{.}\tag{3.14} \end{equation}

In order to see that the above expressions are well defined and have the desired properties, express the curve \(\lambda(t)\) in left-trivialized coordinates as

\begin{equation*} \lambda(t) = \sum_{i=1}^na_i(t)\theta_i(\gamma(t))\text{.} \end{equation*}

The curve \(a(t)\) of (3.13) has the same coefficients as the curve \(\lambda(t)\text{,}\) i.e., it is given by \(a(t) = \sum_{i=1}^na_i(t)\theta_i(e)\text{.}\) Using the explicit expression for the normal Hamiltonian vector field from Lemma 3.3, the normal Hamiltonian ODE \(\dot{\lambda} = \vec{h}_{u(t),-1}(\lambda)\) implies that the components of the curve \(a\) have the derivatives

\begin{equation} \begin{cases} \dot{a}_i(t) = a(t)[u(t),X_i], \amp i\in\{ 1,\ldots, r\}\\ \dot{a}_i(t) = 0, \amp i\in\{ r+1,\ldots, n\} \end{cases}\tag{3.15} \end{equation}

Observe that the vertical coefficients \(a_{r+1},\dots,a_n\) are all constant, and that \(\theta_i([X,Y])=0\) for \(i=1,\ldots,r\text{.}\) Therefore \(a(t)[X,Y] = \sum_{i=r+1}^{n}a_i\theta_i([X,Y])\) is constant in \(t\text{,}\) so (3.14) defines a unique bilinear form \(B\) independent from \(t\text{.}\) Moreover, the non-trivial equations of the system (3.15) are then exactly

\begin{equation*} \dot{a}_i(t) = a(t)[u(t),X_i] = B(u(t),X_i),\quad i\in\{1,\ldots,r\}\text{.} \end{equation*}

Writing an arbitrary vector \(Y\in V_1\) in the basis \(X_1,\ldots,X_r\) as \(Y=y_1X_1+\dots+y_rX_r\text{,}\) the derivative condition 3.1i follows by linearity:

\begin{equation*} \frac{d}{dt}a(t)Y = \frac{d}{dt}\sum_{i=1}^na_i(t)y_i = \sum_{i=1}^nB(u(t),X_i)y_i = B(u(t),Y)\text{.} \end{equation*}

The subdifferential condition 3.1ii for the linear functions \(a(t)\) follows from rephrasing the maximality condition (3.4). Namely, expanding out the explicit expressions of the normal Hamiltonians \(h_{u(t),-1}\) and \(h_{v,-1}\) from (3.1) and reorganizing terms, the maximality condition (3.4) is equivalently stated as

\begin{equation*} a(t)v - a(t)u(t) \leq \frac{1}{2}\norm{v}^2-\frac{1}{2}\norm{u(t)}^2 \quad\forall v\in V_1\quad\text{a.e. }t\in[0,T]\text{.} \end{equation*}

This is exactly Definition 2.16 for the linear function \(a(t)\) being a subdifferential of the squared norm \(\frac{1}{2}\norm{\cdot}^2\) at the point \(u(t)\in V_1\text{.}\)
