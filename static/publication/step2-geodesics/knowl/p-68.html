Indeed, suppose to the contrary that there does not exist a dual curve \((\xi,\lambda)\) with \(\xi\neq 0\text{,}\) but only some dual curve with \(\xi = 0\text{.}\) In this case, the maximality condition (3.4) states that

\begin{equation*} \lambda(t) ( (L_{\gamma(t)})_*u(t) ) \geq \lambda(t) ( (L_{\gamma(t)})_*v )\quad\forall v\in V_1\quad\text{a.e. }t\in[0,T]\text{,} \end{equation*}

which is only possible when \((L_{\gamma(t)})_*V_1\subset\ker\lambda(t)\text{.}\) Moreover, second order optimality conditions from Section 20 of [2] further imply that (possibly changing the dual curve \(\lambda\)) the Goh condition

\begin{equation*} \lambda(t) ((L_{\gamma(t)})_*[v,w])=0,\quad \forall v,w\in V_1\quad\text{a.e. }t\in[0,T] \end{equation*}

is also satisfied. Since the group \(G\) has step 2, its Lie algebra has the decomposition \(\mathfrak{g}=V_1\oplus[V_1,V_1]\text{.}\) The above would then imply that \(\lambda(t)= 0\) almost everywhere, which would contradict the assumption that \((\xi,\lambda)\) is almost everywhere non-zero.
